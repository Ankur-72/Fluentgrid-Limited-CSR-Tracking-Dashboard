# Fluentgrid CSR Intelligence Hub (FY 2026)

## Overview
The Fluentgrid CSR Intelligence Hub is an operational tracking dashboard conceptualized and prototyped during the Rural Immersion for Social Engagement (RISE) internship program. Built to bridge the gap between grassroots field realities and executive strategic monitoring, this tool transforms raw rural data into actionable corporate intelligence.

This dashboard was designed to track the health, logistics, and social Return on Investment (ROI) of highly targeted interventions across rural Visakhapatnam, moving away from static reporting to a dynamic command center model.

## Core Modules & Features

### 1. Asset Health & Infrastructure Matrix
Tracks the operational status and maintenance requirements of physical CSR assets in real time.
* **Gondhuru Tribal RO Plant:** Active monitoring of component health (RO membranes, ATW card readers).
* **Valluru RO Plant:** Critical status alerts and overhaul tracking.
* **Arogya Government Clinic:** Monitoring digital supply chain tablet deployments to prevent critical medicine stockouts.

### 2. Closed-Loop Agritech & Nutrition Pipeline
A visual workflow tracker mapping a dual-impact CSR initiative:
* **Procurement:** Deployment of zero-electricity "Cool Crates" (ColdEasy Boxes) to marginal farmers at Gambheeram and Anandapuram Rythu Bazaars to prevent summer crop spoilage.
* **Distribution:** Routing the preserved agricultural produce directly to the Krupa Sadhana Charitable Trust to secure nutritional needs for orphans.

### 3. Logistics & On-Ground Vendor Directory
A quick-access database to manage field operations without friction.
* Features direct contact matrices for NGO partners, tribal water project leads (e.g., Mr. KV Murthy's community-managed spring models), and verified local transit drivers managing the orphanage school routes.

### 4. Interactive Data Visualization
Embedded analytics to guide executive decision making:
* **Social ROI vs. Financial Outlay:** Bar charts comparing the impact of low-budget interventions (like transit logistics) against heavier infrastructure deployments.
* **Beneficiary Distribution:** Doughnut charts segmenting impact across Scheduled Tribes, orphans, marginal farmers, and rural healthcare patients.

## Tech Stack
* **Markup & Styling:** HTML5, Tailwind CSS (via CDN)
* **Icons:** FontAwesome
* **Data Visualization:** Chart.js (Embedded JavaScript)

## Usage
This is a static, frontend prototype designed for immediate browser execution.
1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser.
3. No build tools or local servers are strictly required to view the UI and interactive charts.
