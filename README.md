# Fluentgrid CSR Intelligence Hub (FY 2026)

Operational CSR dashboard for rural asset health, logistics tracking, agritech workflow monitoring, healthcare readiness, and budget deployment.

## Live Demo
[Open Dashboard](https://ankur-72.github.io/Fluentgrid-Limited-CSR-Tracking-Dashboard/)

## Features
- Real-time style CSR operations dashboard
- Rural asset health monitoring
- Logistics and transport coordination
- Agritech and nutrition workflow tracking
- Vendor and driver directory
- Social ROI and beneficiary distribution charts

## Tech Stack
- HTML
- Tailwind CSS via CDN
- Font Awesome
- Chart.js

## Use Case
Built as a CSR operations and decision-support dashboard for Fluentgrid Limited to help teams monitor field execution and strategic impact in one place.
